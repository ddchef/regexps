## 联系方式系列 
1. 手机号码
```js
/^1\d{10}$/
```
2. 多个手机号码 
```js
/^(1\d{10},?)+$/
```
3. 邮箱
```js
/^[-_a-zA-Z0-9\.]+@[-_a-zA-Z0-9]+(\.[-_a-zA-Z0-9]+)+$/
```
4. 多个邮箱
```js
/^([-_a-zA-Z0-9\.]+@[-_a-zA-Z0-9]+(\.[-_a-zA-Z0-9]+)+,?)+$/
```
